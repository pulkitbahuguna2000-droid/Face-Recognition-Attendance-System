**Face Recognition Attendance System**

**Overview**

This project is a Face Recognition Attendance System built using Python and OpenCV. It automates attendance tracking by detecting and recognizing faces through a live camera feed and recording attendance in a database.

**-->Features**

1. Real-time face detection and recognition
2. Automated attendance marking
3. Dataset generation for training
4. Duplicate attendance prevention
5. SQLite database integration
6. End-to-end attendance management

**-->Technologies Used**

1. Python
2. OpenCV
3. SQLite
4. Haar Cascade Classifier
5. LBPH Face Recognizer

**-->How It Works**

1. Capture facial images and create a dataset.
2. Train the recognition model using LBPH.
3. Detect and recognize faces through the camera feed.
4. Mark attendance automatically in the SQLite database.
5. Prevent duplicate attendance entries.

**Learning Outcomes**

This project helped me gain hands-on experience in computer vision, data preprocessing, model training, database integration, and building a real-world AI application.
